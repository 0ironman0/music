# 甜橙音乐

#### 需求分析:
##### 使用MySQL和flask实现,所有功能需和数据库互联,需要简单网页交互
##### 1.用户注册,登录,退出
##### 2.歌曲分类显示,创建歌曲表时加入一个曲风类型,可以通过曲风分类显示,可以通过语言分类,可以通过歌手分类
##### 3.写一个播放页面,在线听歌,包含歌曲名(歌曲名读取自Python)和播放(或者一个播放按钮),用户点击播放即可听歌
##### 4.排行榜,创建歌曲表时添加播放次数属性,用户每在线听歌一次,播放次数加1,点击排行榜,显示排序后的歌单
##### 5.搜索歌曲,在文本框输入歌曲名进行搜索,显示所有歌名包含关键字的歌曲,点击歌曲名字跳转到播放页面
##### 6.收藏歌曲,创建用户表时添加一个收藏属性,初始为空,用户登录后可以查看和添加收藏(添加外键,关联歌曲表的id),点击可以跳转到播放页面
##### 7.管理员添加歌手及歌曲(歌曲表和歌手表通过外键连接)

### 每个人选一个,简单实现即可,网页不需要图片(想弄随意)
